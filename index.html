What it Iridium?

Iridium is a map editor with focus on mobile devices (android/iOS). Made to be simple to use, and user friendly.

How Iridium works?

The Iridium editor takes the sprites from your local storage at 'android/data/com.oft.iridium/files/
mapName/Tiles'
 and loads all those sprites allowing you create your custom maps with your custom textures. All maps are saved at 'android/data/com.oft.iridium/files/
mapName/map.json'

Functions

Layers:


• t: layer selector mode.
• t + 0: first layer.
• t + 1: back one layer.
• t + 2: pass one layer.
• t + 3: add ten on layer.
• alt: back to map manager.

Objects:


• Ctrl: active the object mode.
• Ctrl + backSpace: active the delete mode to you delete objects.
• Ctrl + 0: first object.
• Ctrl + 1: back ome object.
• Ctrl + 2: pass one object.
• ctrl + 3: add ten on object id.

Blocks:


• Esc: active the block mode.
• Esc + backSpace: active the delete mode to you delete blocks.
• b: back one block.
• n: pass one block

Others Functions


• when you touch/click on screen it mark you touch/click in green.
• arrows: move screen.
• s: save map.
• z/y: undo/redo.
• theme (on the map manager): select the theme
