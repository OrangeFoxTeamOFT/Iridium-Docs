What is the iridium?

The iridium is a map editor with foucus in mobile devices (ANDROID/IOS) made to be simple to use.

How iridium works?

The iridium takes the sprites at:
‘android/data/com.oft.iridium/files/
(mapName)/Tiles/..’
All maps created are saved at:
‘android/data/com.oft.iridium/files/
(mapName)/map.json’.

What ultilities the iridium has?

Blocks:

  • Esc: Active the block mode; it is used to you put blocks.
  • Esc + backSpace: Active the delete mode; it is used to you delete blocks.
  • N: Pass 1 (one) tile; it is used to you select the block texture.
  • B: Back 1 (one) tile; it is used to select the block texture.

Layers:

  • T: Active th layer select mode; it is used to select a layer.
  • T + 0: Go to layer 0 (zero); it is used to show all layers.
  • T + 1: Pass 1 (one) layer; it is used to select a layer.
  • T + 2: Back 1 (one) layer; it is used to selec a layer.
  • T + 3: Pass 10 (ten) layers; it is used to select a layer.
  • T + “−”: Delete a layer; it is used to select a layer.

Objects:

  • Ctrl: Active the objectMode; it is used put objects with ids and select yours ids, can be used to make colission or coin etc.
  • Ctrl + 0: First object id; it is used to select a object id.
  • Ctrl + 1: Pass 1 (one) object id; it is used to select a object id.
  • Ctrl + 2: Back 1 (one) object id; it is used to select a object id.
  • Ctrl + 3: Pass 10 (ten) objects ids; it is used to select a object id.

More functions:

  • Show touch: When you touch on screen it mark you touch in green.
  • Arrows: Move screen; note: it just move in 4 directions.
  • Theme: Select the theme of editor.
  • Z / Y: Undo and Redo; it is used to back ou pass a action.
  • S: Save the map; it is used to save your project.
